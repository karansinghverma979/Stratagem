## 👁️ VISUAL IDENTITIES & DESIGN DIRECTIVE
**Stratagem** is a digital command center designed under the **FUI (Futuristic User Interface)** manifesto. Inspired by military command platforms, quantum interface grids, and tactical cybernetics, it converts routine mission organization, task execution, and focus management into an immersive strategic operation. 

Every component is rendered inside a modular, high-fidelity **volumetric glassmorphism container** with glowing neon accents, moving data sweeps, and real-time hexadecimal telemetry overlays.

## ⚡ CORE CAPABILITIES & SECTORS

Stratagem is organized into dedicated functional zones (Sectors):

### 1. 📂 EXECUTION MATRIX
* **Strategic Mission Registry**: A 100% wide high-contrast grid rendering your primary tactical operations.
* **Intelligent Telemetry**: Mission cards are marked with custom priority indicators (Low, Med, High), category badges, and dynamic status readouts.
* **Dynamic Search & Highlights**: Integrated regex search highlighting matching text in real-time inside task rows.
* **Responsive Mutators**: Smooth cascade waterfall entries and Svelte flip-glide animations on priority and state changes.

### 2. 🧬 ARSENAL BOARD
* **Tactical Kanban Console**: Drag-and-drop board divided into three intelligence tiers: `RAW INTEL`, `SYNTHESIZING`, and `WEAPONIZED`.
* **Weaponize Protocol**: Dragging a task into the `WEAPONIZED` column triggers the **Forge Station** to immediately authorize the task as an active mission.
* **Fluid Mechanics**: Encapsulated HTML5 drag events accompanied by mechanical audio confirmations (`card-drop`).

### 3. 🚨 BREACH SECTOR
* **Critical Compromises**: Isolates and displays failed or compromised missions in a high-intensity **Tactical Red** emergency mode.
* **Security Telemetry**: Styled with stuttering color-split glitch headers, pulsing red vignette backgrounds, and custom warning triggers.
* **Inline Mitigation**: Allows the operator to initiate recovery protocols or purge the target on the fly.

### 4. 🗄️ ARCHIVE HUB
* **Chronological Timeline Log**: Alternating chronological node alignment mapping your historical campaigns (Completed vs. Aborted).
* **Cryptographic Decryption**: Completed mission parameters load utilizing a Svelte-tweened cryptographic decryption animation.
* **After-Action Reports**: Review full parameters, resolution logs, and timeline indicators of archived data.

### 5. ⏱️ CHRONOS SECTOR
* **Temporal Focus Sync**: Pomodoro focus scheduler with glowing SVG circular countdown indicators.
* **Deep Focus Mode**: Locks the interface into a dark CRT raster scanline overlay and floats a persistent, transparent holographic mini-timer visible across all sectors.
* **Background Persistence**: Timer calculations are run through a global store, preventing telemetry loss during sector switching.

### 6. ⚙️ GENESIS SECTOR
* **Core Environment Portals**: Manage system assets including developer profiles, companion nudity toggles, and customized quote books.
* **Mental Skill Trees**: Active nodes detailing personal and cognitive improvements.

### 7. 🛡️ SYSTEM INTELLIGENCE HUB
* **2x2 Command Grid**: Access system logs, live metrics, and database backup routines (Import/Export protocols).
* **Open File Location**: Instantly open database folders directly in Windows Explorer using context IPC bridges.
* **Nuke Purge Safeguard**: A circular 60s countdown trap. Once authorized, it wipes the entire SQLite database, executes hardware-shake animations, and prompts for a system reboot.

## 🚀 INSTALLATION & PRODUCTION DEPLOYMENT

Stratagem v1.0.1 is distributed as a signed, self-contained Windows setup installer executable.

### 1. Execute Setup
* Locate and run the setup file: **[Stratagem-1.0.1-setup.exe](file:///C:/Users/karan/Void/Stratagem/N0_Furnace/dist/Stratagem-1.0.1-setup.exe)**.
* The setup wizard will guide you through the process in a polished, bold interface.

### 2. Configure Directory Portals
During installation, you will be prompted to define target folders for persistent data storage:
1. **Developer Images (`devImages`)**: Saved portrait files.
2. **NoteCards Library (`NoteCards`)**: Custom reference assets.
3. **SQLite Database Folder**: Directory housing the `stratagem_intel.db` state matrix.

> [!NOTE]
> The paths defined in this step are written directly to `HKCU\Software\Strategem 1.0` in the Windows Registry to allow dynamic resolution by the Electron runtime.

## 🧹 UNINSTALLATION CLEANUP PROTOCOL
Should you need to wipe the deployment, running the Windows uninstaller executes a custom macro in our NSIS engine that completely removes:
* The SQLite database directories and raw `.db` databases (if empty).
* Default app data inside `AppData/Roaming/Stratagem` and `n0-furnace`.
* Custom task notes stored at `C:\Users\karan\StratagemNotes`.
* All configuration registry hives (`HKCU\Software\Strategem 1.0`).

> [!IMPORTANT]
> The uninstaller **explicitly preserves** the custom `DeveloperImages` and `NoteCards` asset folders to protect custom media logs.
