# STRATAGEM: N0_FURNACE - Master Architectural Blueprint

## 1. Prime Directives for AI Agents
- **Atomic Isolation:** Every distinct feature, window, or tab MUST be a self-contained Svelte component (`.svelte` file).
- **CSS Scoping:** Never write global CSS. Rely entirely on Svelte's native `<style>` tags to hash and isolate component styling.
- **Portability:** Never use absolute file paths. Always use relative paths or Vite aliases.
- **Update Protocol:** After completing EVERY prompt, you MUST update the "Recent Changes Log" at the bottom of this file.

## 2. Global Design Tokens
The application follows an Industrial Sci-Fi / Tactical Glassmorphism aesthetic. Stick strictly to these hex codes:
- `Base Background`: Deep Obsidian (`#0d0f1e`)
- `Primary Accent`: Electric Purple (`#8b5cf6`)
- `Critical Alert`: Tactical Red (`#ef4444`)
- `Secure Status`: Emerald Green (`#10b981`)
- `Glassmorphism`: `background: rgba(13, 15, 30, 0.6); backdrop-filter: blur(20px);`
- `Fonts`: 'Outfit' for high-impact headers, 'Inter' for UI data.

## 3. Core Folder Matrix (src/renderer/src/)
- `/core/` : Non-UI logic (audio engine, database, internal controller).
- `/components/` : Reusable global UI (Window controls, top tab navigation).
- `/sectors/` : The completely isolated feature zones (Execution, Chronos, Genesis, etc.).
